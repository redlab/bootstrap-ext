Typeaheadmap

A make over of bootstraps typeahead that supports an array of objects, a listener function hook to receive selected data, and a customizable displayer function

In this example you can start typing a country in the country box, see the dropdown with the country and capital. Upon selection, the capital is set in the capital input box. The displaying and list size is altered from the defaults of typeaheadmap.(Same goes for the capitalbox)
